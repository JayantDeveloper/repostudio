# Background Music Tracks

Drop royalty-free instrumental MP3s here (no vocals):

| File | Mood | Where to source |
|---|---|---|
| `cinematic.mp3` | Epic/orchestral | Pixabay, Freepd.com |
| `upbeat.mp3` | Electronic/energetic | Pixabay, ccmixter.org |
| `minimal.mp3` | Ambient/calm | Freepd.com, Free Music Archive |
| `hype.mp3` | High-energy/punchy | Pixabay, ccmixter.org |

Or override via env vars:
```
NEXT_PUBLIC_MUSIC_CINEMATIC_URL=https://...
NEXT_PUBLIC_MUSIC_UPBEAT_URL=https://...
NEXT_PUBLIC_MUSIC_MINIMAL_URL=https://...
NEXT_PUBLIC_MUSIC_HYPE_URL=https://...
```

All tracks are ducked to ~8% volume during narration, fading to ~28% in gaps.
